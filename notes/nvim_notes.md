## NOTES


## COMMANDS
- `SPACE + e` -> toggle side bar file tree
- `: + e "file name" ` -> creates a new file
- `: w` -> saves the file
- `NORMAL + number + (g-j-h)` -> jump the number of the lines to the given direction.


## VISUAL MODE
- `v` -> start visual mode
- `esc` -> back to normal
- copy and past: `y` yanks the text `p` to paste 
- 

## NORMAL MODE NAVIGATION

- `j` -> down 
- `g` -> up
